### 작동 방식, 사용법
1. 중심 위도, 경도 설정
2. 배율 설정
    -   배율은 총 3가지
    -   1-3단계가 순서대로 5-7의 상수값을 가짐
    -   이 값은 높이, 너비의 중심 좌표가 됨
    -   Ex. 1단계가 2의 상수값을 갖는다고 가정한다면
        -   ㅡㅡㅡㅡㅡㅡㅡㅡㅡㅡㅡㅡㅡ
        -   | (0, 0) (0, 1) (0, 2) (0, 3) (0, 4) | 
        -   | (1, 0) (1, 1) (1, 2) (1, 3) (1, 4) |
        -   | (2, 0) (2, 1) (2, 2) (2, 3) (2, 4) |
        -   | (3, 0) (3, 1) (3, 2) (3, 3) (3, 4) | 
        -   | (4, 0) (4, 1) (4, 2) (4, 3) (4, 4) |
        -   ㅡㅡㅡㅡㅡㅡㅡㅡㅡㅡㅡㅡㅡ
3. 사진 조립
    -   (0, 0)에 해당하는 좌표를 찾은 후
    -   차례대로 사진을 얻어옴   
    -   가져온 사진은 Canvas에 조립
